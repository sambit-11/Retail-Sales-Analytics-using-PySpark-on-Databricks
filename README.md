# Retail Sales Analytics using PySpark on Databricks

## Overview

This project demonstrates retail sales data analysis using PySpark on Databricks. The objective was to clean, transform, and analyze retail transaction data to generate meaningful business insights. The project leverages PySpark DataFrame APIs for distributed data processing and concludes with an interactive dashboard that visualizes key retail performance metrics.

## Objectives

* Import and process retail sales datasets.
* Define custom schemas for structured data loading.
* Clean and transform the data using PySpark.
* Perform joins and aggregations to generate business insights.
* Develop an interactive dashboard for sales analysis.

## Technologies Used

* Python
* PySpark
* Apache Spark
* Databricks
* Git & GitHub

## Dataset

The project uses three datasets:

* Sales – Customer order transactions
* Menu – Product details and food categories

## Data Processing

The following operations were performed using PySpark:

* Data ingestion
* Schema definition
* Data cleaning
* Data transformation
* Filtering and sorting
* Joins
* GroupBy operations
* Aggregations
* Business analysis

## Dashboard KPIs

The dashboard provides insights into:

* Annual Sales
* Quarterly Sales
* Top 5 Ordered Items
* Total Amount Spent by Each Food Category
* Customer Visit Frequency
* Sales by Order Source
* Number of Orders by Each Food Category
* Total Amount Spent by Each Customer
* Total Sales Generated by Each Customer
